ПРИМЕЧАНИЕ: Basic-версия клиента F-Droid имеет урезанный набор функций (например, нет общего доступа к близлежащим ресурсам и функции паники). Она нацелена на Android 13 и может выполнять обновления без участия пользователя без привилегированного расширения или root.

F-Droid — это каталог свободных приложений для Android. Клиентское приложение F-Droid позволяет легко просматривать, устанавливать и отслеживать обновления на вашем устройстве.

Оно подключается к любым репозиториям, совместимым с F-Droid. Репозиторий по умолчанию находится на сайте f-droid.org, который содержит только подлинно свободное программное обеспечение.

Сам Android открыт в том смысле, что вы можете свободно устанавливать APK из любого места, но есть много веских причин использовать F-Droid, в качестве свободного менеджер приложений:

* Получать уведомления о выходе обновлений.
* Возможность автоматической загрузки и установки обновлений
* Отслеживание устаревших и бета-версий
* Отфильтровать приложения, несовместимые с устройством.
* Производить поиск приложения по их категориям и содержимому их описаний
* Доступ к связанным URL-адресам: для пожертвований, исходного кода и т.д.
* Обеспечение собственной безопасности, проверяя подписи индексов репозитория и хешей APK
