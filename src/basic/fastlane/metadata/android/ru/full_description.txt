ПРИМЕЧАНИЕ: Базовая версия F-Droid Client имеет урезанный набор функций (например, нет общего доступа к близлежащим ресурсам и функции паники). Она нацелена на Android 13 и может выполнять обновления без участия пользователя без привилегированного расширения или root.


F-Droid - это устанавливаемый каталог свободных программ для Android. Клиентское приложение F-Droid позволяет легко просматривать, устанавливать и отслеживать обновления на вашем устройстве.


Оно подключается к любым репозиториям, совместимым с F-Droid. Репозиторий по умолчанию находится на сайте f-droid.org, который содержит только добросовестное свободное программное обеспечение.


Сам Android открыт в том смысле, что вы можете свободно устанавливать APK из любого места, но есть много веских причин использовать F-Droid в качестве менеджера приложений libre:


* Получать уведомления о выходе обновлений.

* Возможность автоматической загрузки и установки обновлений

* Отслеживайте старые и бета-версии

* Отфильтровать приложения, несовместимые с устройством.

* Находите приложения по категориям и описаниям с возможностью поиска

* Доступ к связанным URL-адресам для пожертвований, исходного кода и т. д.

* Обеспечьте безопасность, проверяя подписи индексов репо и хэши APK
